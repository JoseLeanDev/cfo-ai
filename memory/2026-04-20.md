# Memory - 2026-04-20

## 🔥 URGENTE: Agentes no funcionaban - ROOT CAUSE ENCONTRADO

**Problema reportado por usuario:** "Los agentes SIGUEN SIN FUNCIONAR"

### Diagnóstico
Verifiqué los endpoints:
- `GET /api/agents/status` → `{ isRunning: false, message: "Agentes no inicializados aún" }`
- `GET /api/agents/ia/status` → `{ estado: "no_inicializado", mensaje: "Agentes de IA no iniciados" }`

### Causas raíz encontradas

#### 1. Agentes de IA no inicializados
- **Causa:** `render.yaml` no tenía la variable `OPENROUTER_API_KEY`
- **Efecto:** El código en `src/index.js` verifica `if (process.env.OPENROUTER_API_KEY)` y si no existe, no inicia los agentes
- **Fix:** Agregué `OPENROUTER_API_KEY`, `LLM_PROVIDER`, y `APP_URL` a `render.yaml`

#### 2. Agentes tradicionales no inicializados
- **Causa:** Import incorrecto en `src/index.js`
  - Tenía: `const AgentOrchestrator = require('../agents');` 
  - Apuntaba a `backend/agents/index.js` (que es el orquestador de IA)
  - Debería ser: `const { getOrchestrator } = require('./agents');`
  - Que apunta a `backend/src/agents/index.js` (orquestador tradicional)
- **Efecto:** Usaba el orquestador de IA en lugar del tradicional, y con `new AgentOrchestrator(db)` que no existe
- **Fix:** Corregí el import y usé `initializeOrchestrator()` en vez de `new AgentOrchestrator(db)`

#### 3. Método faltante en orquestador tradicional
- **Causa:** La ruta `/api/agents/status` llamaba `orchestrator.getStatus()` pero el orquestador tradicional solo tenía `getSystemStatus()`
- **Fix:** Agregué método `getStatus()` que es compatible con la API

#### 4. ERROR CRÍTICO: "AgenteClass is not a constructor"
**Reportado por usuario con screenshot de logs**
- **Causa:** Los 4 archivos de agentes IA exportaban instancias en lugar de clases:
  ```javascript
  module.exports = new AuditorAgentIA();  // ❌ Exporta instancia
  module.exports = AuditorAgentIA;        // ✅ Exporta clase
  ```
- **Efecto:** El orquestador hace `new AgenteClass()` pero recibe una instancia ya creada, no un constructor
- **Fix:** Cambié los 4 archivos para exportar la clase:
  - `AuditorAgentIA.js`
  - `AnalistaFinancieroIA.js`
  - `ConciliadorAgentIA.js`
  - `MaintenanceAgentIA.js`
