# CFO AI Project - Fixes del 15 de Abril 2026

## Problemas resueltos hoy

### 1. Insights de IA no funcionaban
**Causa raíz:** El endpoint `/analisis/insights` usaba `empresa_id = 'default'` mientras que la BD tenía `empresa_id = 1`.

**Fix:** Cambiar default de `'default'` a `1` en:
- `backend/src/routes/analisis.js`
- `backend/src/routes/agents.js`
- `backend/src/services/agentLogger.js`

### 2. Insights contextuales por área
**Implementación:** Cada insight ahora tiene `contexts` (array) para saber en qué páginas mostrarse:
- `tesoreria`: CxC/CxP vencidas, flujo de caja
- `contabilidad`: Gastos, márgenes, transacciones
- `analisis`: Clientes, tendencias, proyecciones
- `dashboard`: Todos (resumen ejecutivo)

**Commits:** `14c1684`, `25ef543`

### 3. Runway Calculator mostraba datos incorrectos
**Problemas:**
- Usaba valores hardcodeados (Q2.2M gastos, Q2.5M ingresos)
- Mostraba "∞" para empresas rentables
- Gráfica vacía o con barras muy bajas

**Fixes aplicados:**
- Backend calcula promedios reales de últimos 6 meses
- Muestra "Sin límite" + "Duplicas efectivo en X meses" en lugar de ∞
- Simplificado cálculo de barras: `(saldo / maxSaldo) * 100`

### 4. Logs transformados a lenguaje de negocio
**Cambios:**
- Orchestrator: eliminados logs técnicos de routing
- Auditor IA: logs reescritos con mensajes accionables
- Analista Financiero: agregado logging en todos los análisis
