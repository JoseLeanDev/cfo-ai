# Opciones para Backend Permanente

## 1. 🥇 Cloudflare Tunnel (GRATIS + Fijo)
- Dominio fijo gratuito (tudominio.trycloudflare.com)
- No expira
- Gratis para uso personal
- Requiere instalar cloudflared

## 2. 🥈 Render.com (GRATIS)
- Deploy automático desde GitHub
- URL fija: tudominio.onrender.com
- Free tier: se duerme después de inactividad (tarda ~30s en despertar)
- SQLite no persiste entre reinicios (necesitarías PostgreSQL)

## 3. 🥉 Railway.app (GRATIS $5/mes)
- Deploy simple desde GitHub
- Base de datos persistente
- $5 de crédito mensual (suficiente para proyecto pequeño)

## 4. VPS Propio (Si tienes uno)
- Dominio fijo
- Backend siempre activo
- Requiere servidor propio

---

## Recomendación: Cloudflare Tunnel

Es gratis, fácil de configurar y da una URL permanente.
